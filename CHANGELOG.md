# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]
## [0.2.1] - 2026-04-17

### Changed
- Renamed `backend` skill to `clean-architecture-ddd-nestjs` to match its frontmatter name and improve discoverability.

## [0.2.0] - 2026-04-17

### Added
- New `clean-architecture-ddd-nestjs` skill: enforces Clean Architecture with DDD-inspired modeling for NestJS backends, covering layer responsibilities, domain modeling, use case rules, Prisma governance, and testing constitution.

## [0.1.2] - 2026-04-16

### Changed
- Simplified `create-pr` documentation into a single `SKILL.md` file and removed redundant reference files.

## [0.1.1] - 2026-04-16

### Added
- Changelog structure created to track future project updates.
- Root `README.md` added with repository overview and guidelines for adding new skills.

### Changed
- Updated `create-pr` skill workflow/spec to better handle missing tooling and clarify PR publishing steps.

## [0.1.0] - 2026-04-16

### Added
- Initial `create-pr` skill:
  - `create-pr/SKILL.md` with end-to-end publish workflow
  - `create-pr/README.md` with usage and installation instructions
  - `create-pr/reference.md` with execution priorities and failure handling
